Netflix KPIs Dashboard
A Tableau dashboard designed to explore key performance indicators across the Netflix catalog. The project focuses on understanding content performance, category trends, and the distribution of titles across countries and genres. It highlights how data visualization can support strategic decisions in content analysis.
Project Overview
- Analysis of content performance and catalog composition
- Trends across genres, categories, and release years
- Geographic distribution of Netflix titles
- KPI-driven insights to understand platform content strategy

Dataset
The dataset includes publicly available information about Netflix titles, containing fields such as:
- Title, type, and release year
- Country of origin
- Genre and category
- Rating and additional metadata
The CSV file used for this project is included in this repository.
Tools Used
- Tableau for data visualization
- CSV preprocessing for data cleaning and preparation
- Visual storytelling techniques to highlight KPIs and insights
